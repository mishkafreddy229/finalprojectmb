Тип моего проекта:
Telegram-бот для экопросвещения и борьбы с глобальным потеплением - интерактивная платформа, которая сочетает образовательные материалы, практические инструменты и мотивационные элементы. Бот будет помогать пользователям понимать проблему изменения климата, отслеживать свой экологический след, получать персонализированные рекомендации и участвовать в экологических челленджах.

Библиотеки, которые я буду использовать
telebot - для телграма основа

python-dotenv - для управления конфигурацией через .env файлы

Референсы, которые мне пригодятся
@EcoChangeBot - бот с ежедневными эко-советами и челленджами

@GreenBotAssistant - бот, который помогает находить точки сдачи вторсырья

@ClimateActionBot - бот отслеживания климатических обязательств стран

Carbon Footprint Calculator (web) - для вдохновения по расчету углеродного следа

WWF Ecological Footprint Calculator - как эталон расчета экоследа

UN Climate Action - для структурирования образовательного контента

NASA Climate Change - для научных данных и визуализаций

Гайды-статьи, где есть полезная для меня информация

Также ии немножко поможет
